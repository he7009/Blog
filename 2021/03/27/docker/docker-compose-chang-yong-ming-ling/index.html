docker-compose.yml 文件部分命令说明

build 指定 Dockerfile 所在文件夹路径
container_name 指定容器名称
depends_on 指定多个服务之间的依赖关系，启动时会先启动被依赖服务
entrypoint 覆盖容器中默认的入口命令
environment 设置环境变量
image 指定为镜像名称或镜像ID，如果本地镜像不存在，Compose会尝试拉取这个镜像
networks 所加入的网络
ports 端口映射
volumes 数据卷路径映射
restart 设置重启策略
